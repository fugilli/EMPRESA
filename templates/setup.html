Ficheiro credentials.json não encontrado na pasta da app.

Segue estes passos para obteres as credenciais do Google Calendar:

  1. Vai a console.cloud.google.com
  2. Cria um projeto novo (ou usa um existente)
  3. Activa a Google Calendar API
    • Menu lateral → APIs e serviços → Biblioteca → pesquisa "Google Calendar API" → Ativar
  4. Cria credenciais OAuth 2.0
    • APIs e serviços → Credenciais → Criar credenciais → ID de cliente OAuth
    • Tipo de aplicação: Aplicação de computador
    • Dá um nome (ex: "Gestão Empresa")
  5. Faz download do JSON e guarda-o como credentials.json dentro desta pasta: 
    /Users/gilfigueiredo/EMPRESA/credentials.json
  6. No ecrã de consentimento OAuth, adiciona o teu e-mail como utilizador de teste
  7. Depois de guardares o ficheiro, recarrega esta página.
